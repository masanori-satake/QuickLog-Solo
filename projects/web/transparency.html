透明性レポート

私のデータは安全ですか？（どこに保存されますか？）

はい、安全です。データは外部サーバーではなく、あなたのPC上のブラウザ内（IndexedDB）にのみ保存されます。私たちはあなたのデータを見ることができません。

verified サーバー通信（API送信）を行うコードを一切含みません。
verified CSP（Content Security Policy）により、意図しない外部への通信を技術的に遮断しています。

行動を監視（トラッキング）されていませんか？

いいえ。Google Analyticsなどのアクセス解析や、トラッキング目的のコードは1行も含まれていません。あなたが「いつ」「どのカテゴリ」を記録したかは、あなただけの秘密です。

verified プロダクション依存関係（外部ライブラリ）はゼロです。
verified ビルドツールによる難読化を行わず、人間が読めるソースコードのまま提供しています。

10年後もこのツールは使えますか？

私たちは「一生モノの設計」を目指しています。特定のフレームワークに依存せず、標準的なWeb技術（Vanilla JS）のみで構築されているため、ブラウザがある限り動作し続けます。

verified 外部サービスが終了しても、このツールが使えなくなることはありません。
verified 全ソースコードがオープンソースとして公開されており、透明性が確保されています。

セキュリティの脆弱性は大丈夫ですか？

自動化された監査ツールを使用し、常に安全性をチェックしています。

verified Google OSV-Scanner により、開発環境の脆弱性を継続的に監査しています。
verified GitHub Actions により、すべての変更に対してセキュリティ・整合性チェックを自動実行しています。
history_edu 実際の監査ログを確認する（GitHub Actions）
