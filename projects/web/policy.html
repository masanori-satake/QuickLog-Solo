プライバシーポリシー

データの所在

本ソフトウェアで記録されるすべてのデータ（作業ログ、カテゴリ設定、アプリケーション設定）は、利用者のPC上のブラウザ内（IndexedDB）にのみ保存されます。開発者を含む第三者がこれらのデータにアクセスすることは技術的に不可能です。

通信とトラッキング

本ソフトウェアは、アクセス解析、トラッキング、広告表示、またはデータの外部送信を行うコードを一切含みません。完全なオフライン（ローカル）での動作を基本としています。

バックアップ

利用者が明示的に「バックアップ」機能を実行した場合に限り、ブラウザの File System Access API を通じて、利用者が指定したローカルフォルダにデータが書き出されます。この際も、外部サーバーを介することはありません。
