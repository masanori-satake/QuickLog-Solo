# QuickLog-Solo ガイドブック (開発・設定リファレンス)

本ドキュメントは、QuickLog-Solo の補助ツールである **QL-Category Editor** および **QL-Animation Studio** の操作方法と詳細な仕様をまとめたガイドです。初めて利用するユーザーがスムーズに操作でき、開発者がオンラインガイドを構築する際のリファレンスとなることを目的としています。

---

## 1. はじめに

QuickLog-Solo は、ユーザーが作業スタイルに合わせて記録環境を自由にカスタマイズできる設計思想を持っています。

- **QL-Category Editor**: カテゴリの一括編集、高度なタグ管理、並べ替えを広い画面で行うためのツール。
- **QL-Animation Studio**: 背景アニメーションを自ら作成・調整・テストするための開発環境。

これらのツールはウェブ上で動作し、データ連携は「クリップボード経由の NDJSON 形式」で行います。

---

## 2. QL-Category Editor 操作ガイド

カテゴリ（作業項目）の名称、色、アニメーション、タグを一括で管理します。

### 2.1. カテゴリリストの操作
- **追加と削除**:
    - **カテゴリ追加**: ヘッダーの `+` ボタンをクリック。
    - **ページ区切り追加**: リスト下部の「改ページを追加 (Add Page Break)」ボタンをクリック。
    - **削除**: 各項目のゴミ箱アイコン、または `more_vert` メニューから実行。
- **並べ替え**: 左端の `drag_indicator` をドラッグ＆ドロップ。移動先が青いライン（インジケーター）で表示されます。
- **複製**: `more_vert` メニューの「複製」を選択。名称に `(1)` などの連番が自動付与されます。

### 2.2. 高度な選択操作
ファイルエクスプローラーのような直感的な選択が可能です。
- **単一選択**: 項目をクリック。
- **複数選択 (個別)**: `Ctrl` (Macは `Cmd`) を押しながらクリック。
- **範囲選択**: 1つ選択後、`Shift` を押しながら別の項目をクリック。

### 2.3. 詳細編集パネルの挙動
- **複数選択時の動作**:
    - **名称**: 編集不可（一括で名前を変えることはできません）。
    - **タグ**: 選択されたすべてのカテゴリに**共通して含まれるタグ（積集合）**のみが表示されます。
    - **色・アニメーション**: 最初の一つの値が表示されます。変更すると選択中の全カテゴリに一括適用されます。
- **タグの入力**: 文字入力後に `Enter` または `,`（カンマ）で確定。
- **Tag Box (タグボックス)**: 全カテゴリで使用中のユニークなタグを一覧表示。
    - **ドラッグ＆ドロップ**: タグをボックスからタグ入力欄へドラッグして追加できます。ドラッグ中は入力欄が点線で強調されます。
- **改ページ (Page Break)**: 名称や色、アニメーション、タグを持ちません。リスト上での並べ替え専用の項目です。

### 2.4. Undo/Redo (履歴管理)
最大50手まで遡ることが可能です。
- **テキスト編集のグループ化**: 名前の変更などは、入力開始から完了（フォーカスアウト）までを1つのアクションとして記録します。
- **ショートカット**:
    - `Ctrl + Z` (Mac: `Cmd + Z`): 元に戻す。
    - `Ctrl + Y` または `Ctrl + Shift + Z` (Mac: `Cmd + Shift + Z`): やり直し。

## 3. QL-Animation Studio 操作ガイド

背景アニメーションをゼロから構築、または既存サンプルを調整するための環境です。

### 3.1. 構成要素
- **Metadata**: 名前、作者、説明。多言語対応しており、言語を切り替えてそれぞれの翻訳を登録可能。
- **Configuration**:
    - **Mode**: 描画エンジン（Canvas 2D / Sprite / Matrix）の選択。
    - **Exclusion Strategy**: UI要素を避けるアルゴリズム（`mask`: 隠す, `jump`: 飛び越える, `freedom`: 無視）。
    - **Rewindable**: タイムトラベル（巻き戻し）対応の有無。

### 3.2. スマートエディタ機能
JavaScript のコーディングを支援する機能が搭載されています。
- **オートインデント**: `Enter` キーで改行した際、前の行のインデントを自動で引き継ぎます。
- **オートクローズ**: `{`, `[`, `(`, `"`, `'` を入力すると、対になる閉じ記号を自動挿入します。
- **検索・置換**: `Ctrl + F` (Mac: `Cmd + F`) で検索バーを表示し、現在のタブ内の文字列を検索・一括置換できます。
- **折り返し切り替え**: ツールバーのアイコンでコードの折り返し（Word Wrap）を ON/OFF できます。

### 3.3. テスト走行 (テープデッキ操作)
カセットデッキを模したインターフェースで動作を確認します。
- **再生 (▶)**: コードをコンパイルし実行。
- **一時停止 (⏸)**: 描画を停止。
- **停止 (■)**: 完全に終了し、コード編集モードへ。
- **速度調整**: 0.5x 〜 2.0x のレンジで調整可能。
- **エジェクト (⏏)**: スタジオの状態を初期化（サンプル選択解除）。

### 3.4. シミュレーターと計測
- **Exclusion Area**: プレビュー上の四角形をドラッグ・リサイズして、時計などのUI要素を避ける動きをテスト。
- **Metrics (計測)**:
    - **Latency**: 描画にかかった時間（ms）。
    - **Density**: 描画の密度。
    - **Change Rate**: 前フレームからの変化率。
- **Console**: デバッグ出力や実行時エラーを表示。

## 4. 本体アプリ (QuickLog-Solo) とのデータ連携

### 4.1. インポート/エクスポートの手順
1. **Editor 側でコピー**: 右下の「コピー (Export)」ボタンを押すと、NDJSON形式のデータがクリップボードに保存されます。
2. **本体側で設定を開く**: ⚙️ボタン ＞ 「カテゴリ」タブへ。
3. **インポート実行**: 「貼り付け (Import)」ボタンをクリック。
    - **追記 (Append)**: 現在のリストの末尾に追加。
    - **上書き (Overwrite)**: 現在のリストを空にしてから追加。
4. **視覚的フィードバック**: インポート中はボタンが反応しなくなり、ローディングが表示されます。完了後、500msの意図的な遅延を経て成功通知（トースト）が表示されます。

### 4.2. スキーマバリデーション
データ連携時は厳格なスキーマチェックが行われます。
- `kind` (例: `QuickLogSolo/Category`) や `version` (例: `1.0`) が不足している、または型が異なるデータは拒否されます。
- 一部の行のみが不正な場合、ユーザーに「正常な分だけ取り込むか、中止するか」の選択肢が提示されます。

## 6. 注意事項

- **上書きの警告**: 本体へのインポート（上書きモード）は取り消しができません。事前に本体側で「エクスポート（コピー）」を行い、バックアップを取ることを推奨します。
- **アニメーション名**: かつて「Matrix Code」と呼ばれていたアニメーションは「Digital Rain (`digital_rain`)」に名称変更されました。古い形式のデータはインポート時に自動的に読み替えられます。
- **開発者モード (Studio)**: 作成したアニメーションを正式にアプリに組み込むには、GitHub でのプルリクエストが必要です。「Prepare PR」ボタンから手順を確認してください。
