QuickLog-Solo

1秒で記録、1秒で集計。

QuickLog-Soloは、業務記録を負担に感じるが、ツールの透明性や安全性に厳しい技術者のために設計された、ミニマリスト向けサイドバー型工数管理ツールです。

Chrome / Edge 用 (.zip) Firefox 用 (.zip)

大事なポリシー

lock

Local Only

データはブラウザ内の IndexedDB にのみ保存。外部サーバーへの送信は一切行いません。

visibility_off

No Tracking

アクセス解析やトラッキングコードは一切含みません。あなたの作業を誰も監視しません。

speed

Zero Friction

「Less is More」。過剰な機能を削ぎ落とし、1秒で打刻できる軽快さを追求しました。

  • bolt

    ワンクリック打刻

インストール / アンインストール

インストール方法

  1. ZIPファイルをダウンロードして解凍します。
  2. ブラウザの拡張機能管理ページのURLをコピーして開きます。
    3. (Chrome: chrome://extensions/ content_copy / Edge: edge://extensions/ content_copy)
  3. 「デベロッパーモード」をONにします。
  4. 「パッケージ化されていない拡張機能を読み込む」を選択し、解凍したフォルダを選択します。
  1. ZIPファイルをダウンロードして解凍します。
  2. FirefoxのデバッグページのURLをコピーして開きます。
    about:debugging content_copy
  3. 「一時的な拡張機能を読み込む...」をクリックします。
  4. 解凍したフォルダ内の manifest.json を選択します。

※ Firefoxの「一時的な拡張機能」はブラウザ再起動時にリセットされるため、継続利用にはアドオンストアからの配布（今後予定）または開発者設定が必要です。

※ 外部サーバーを介さないため、安心してお使いいただけます。

アンインストール方法

拡張機能管理画面から「削除」を選択するだけです。IndexedDBに保存されたデータもブラウザの仕様に従い削除されます。

FAQ / トラブルシューティング

Q. データはどこに保存されますか？

A. あなたのPC内のブラウザが管理する領域（IndexedDB）にのみ保存されます。

Q. PCを再起動してもデータは消えませんか？

A. はい、消えません。ブラウザのストレージ機能により、永続的に保存されます。ただし、ブラウザの「閲覧履歴の消去」でデータを削除した場合は消去される可能性があります。

Q. 動作が重くなった場合は？

A. 設定の「メンテナンス」タブからログをクリアするか、CSVエクスポートでバックアップを取ってからリセットをお試しください。
