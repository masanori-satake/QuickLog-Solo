QuickLog-Solo

1秒で記録、1秒で集計。

QuickLog-Soloは、業務記録を負担に感じるが、ツールの透明性や安全性に厳しい技術者のために設計された、ミニマリスト向けサイドバー型作業メモツールです。

リリース版
Chrome / Edge 用 (.zip) Firefox 用 (.zip)
開発版
Chrome / Edge 用 (Dev) (.zip) Firefox 用 (Dev) (.zip)
※過去のバージョンはGitHubリリースページから入手可能です。

大事なポリシー

lock

Local Only

データはブラウザ内の IndexedDB に保存され（バックアップを実行した際にはローカルファイルシステムにも保存されます）、外部サーバーへの送信は一切行われません。

visibility_off

No Tracking

アクセス解析やトラッキングコードは一切含みません。あなたの作業を誰も監視しません。

hard_drive

Local Backup

ローカルフォルダへのバックアップに対応。ブラウザのキャッシュ消去に備えた安心設計です。バックアップデータがあれば、他のブラウザへの移行もスムーズに行えます。

update

Pure JavaScript

外部ライブラリを使わず「JavaScriptのみ」で構築。10年後も安心して使い続けられます。

主な機能

  • bolt

    1秒で記録、1秒で集計

    カテゴリを選ぶだけで即座に開始。日報や集計データもワンクリックで作成できます。

  • side_navigation

    サイドバー常駐

    ブラウザのサイドパネルで動作。作業を妨げず、いつでもブラウザの傍らでクイックに記録可能です。

  • auto_fix_high

    Visual Healing

    20種類以上のドットアニメーションが「1秒の重み」を表現。作業中のストレスを軽減します。

  • analytics

    タグ別集計・自動記録

    プロジェクト横断の集計や、深夜の日付変更時の”作業終了"自動記録など、実用的な機能を備えています。

FAQ / トラブルシューティング

Q. データはどこに保存されますか？

A. あなたのPC内のブラウザが管理する領域（IndexedDB）に保存されます（バックアップを実行した際には、ローカルファイルシステムにも保存されます）。

Q. PCを再起動してもデータは消えませんか？

A. はい、消えません。ブラウザのストレージ機能により、永続的に保存されます。ただし、ブラウザの「閲覧履歴の消去」などでデータが消去される可能性があるため、設定の「バックアップ」タブから定期的にバックアップを実行することを強く推奨します。

Q. 動作が重くなった場合は？

A. 設定の「メンテナンス」タブからログをクリアするか、CSVエクスポートでバックアップを取ってからリセットをお試しください。

Q. サイドバーが狭くて、業務カテゴリの編集が大変です。

A. 広い画面で効率よく編集できる専用のエディタを用意しています。並べ替えや色の確認もスムーズに行えます。

Q. 不具合を見つけましたがどのように報告すべきですか？

A. GitHubのIssueページから報告いただけます。バグ報告用のテンプレートを用意していますので、そちらから詳細をお知らせください。
