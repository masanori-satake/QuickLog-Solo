QuickLog-Solo

1秒で記録、1秒で集計。1秒で安心。

QuickLog-Soloは、業務記録を「極限まで楽に、少しだけ楽しく」することを目指した、ミニマリスト向けサイドバー型作業メモツールです。

Chrome / Edge 用 (.zip) Firefox 用 (.zip)

大事なポリシー

lock

Local Only

データはブラウザ内の IndexedDB にのみ保存。外部サーバーへの送信は一切行いません。

cleaning_services

No Pollution

DLLやレジストリを汚しません。削除すれば跡形もなく消えるクリーンな設計です。

speed

Zero Friction

「前の作業を止める」手間すら省く自動終了ロジック。作業の勢いを止めません。

インストール / アンインストール

インストール方法

  1. ZIPファイルをダウンロードして解凍します。
  2. ブラウザの拡張機能管理ページのURLをコピーして開きます。
    3. (Chrome: chrome://extensions/ content_copy / Edge: edge://extensions/ content_copy)
  3. 「デベロッパーモード」をONにします。
  4. 「パッケージ化されていない拡張機能を読み込む」を選択し、解凍したフォルダを選択します。
  1. ZIPファイルをダウンロードして解凍します。
  2. FirefoxのデバッグページのURLをコピーして開きます。
    about:debugging content_copy
  3. 「一時的な拡張機能を読み込む...」をクリックします。
  4. 解凍したフォルダ内の manifest.json を選択します。

※ Firefoxの「一時的な拡張機能」はブラウザ再起動時にリセットされるため、継続利用にはアドオンストアからの配布（今後予定）または開発者設定が必要です。

※ 外部サーバーを介さないため、安心してお使いいただけます。

アンインストール方法

拡張機能管理画面から「削除」を選択するだけです。IndexedDBに保存されたデータもブラウザの仕様に従い削除されます。

FAQ / トラブルシューティング

Q. データはどこに保存されますか？

A. あなたのPC内のブラウザが管理する領域（IndexedDB）にのみ保存されます。

Q. PCを再起動してもデータは消えませんか？

A. はい、消えません。ブラウザのストレージ機能により、永続的に保存されます。ただし、ブラウザの「閲覧履歴の消去」でデータを削除した場合は消去される可能性があります。

Q. 動作が重くなった場合は？

A. 設定の「メンテナンス」タブからログをクリアするか、CSVエクスポートでバックアップを取ってからリセットをお試しください。

機能の詳細

auto_stop 自動終了ロジック

「開始」を押すだけで、それまで行っていた作業は自動的に終了処理されます。計測を止めるという余計な意識を捨てて、次の仕事に1秒でフォーカスできます。

analytics Auto-stop visualization placeholder

auto_awesome 癒やしのアニメーション

20種類以上のドットアニメーション（Hero Pot, Cats, Digital Rain等）が、作業の傍らで静かに流れます。無機質になりがちな業務ツールを、愛着の持てる相棒に変えてくれます。

pets Animations preview (Cats, Rain, etc.) placeholder

description 高度な日報出力

MarkdownやHTMLテーブル、CSVなど多彩な形式でコピー可能。時刻アジャスト（丸め）機能を使えば、一日の終わりに行う面倒な集計作業が一瞬で終わります。

content_paste Report formats preview placeholder

label タグ集計

カテゴリごとに設定したタグ（プロジェクト名や作業種別）を集計。複数のカテゴリを跨いで「どのプロジェクトに合計何時間使ったか」を簡単に把握できます。

rocket_launch 今後の予定

ブラウザ上で誰でもアニメーションを自作・試行錯誤できる「AnimStudio」を計画中。あなたの作ったアニメーションが、世界中のエンジニアの集中と癒やしを支える日が来るかもしれません。
