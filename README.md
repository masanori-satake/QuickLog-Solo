# QuickLog-Solo

「1秒で記録、1秒で集計、1秒で安心」をコンセプトにした、ミニマリスト向け・完結型工数管理ツールです。

## 特徴
- **完全ローカル動作:** 記録されたデータはすべてブラウザ内の IndexedDB に保存されます。外部サーバーへの送信は一切行われません。
- **PWA (Progressive Web App):** ブラウザから「インストール」することで、デスクトップアプリのように利用可能です。
- **自動終了ロジック:** 新しいタスクを開始すると、自動的に前のタスクを終了します。
- **データ管理:** 30日を過ぎたデータは自動的にクリーンアップされ、メンテナンスフリーで利用できます。

## インストール・使用方法
### ブラウザでの利用
1. `index.html` をブラウザで開きます。
2. GitHub Pages 等にホストされている場合は、アドレスバー右側の「インストール」アイコンからデスクトップに追加できます。

### データの保存場所
- Google Chrome/Edge/Firefox 等のブラウザ内ストレージ（IndexedDB）に保存されます。
- ブラウザのキャッシュやデータを完全に削除すると、記録も消去されるためご注意ください。定期的なCSVエクスポートを推奨します。

## トラブルシューティング
### アプリが起動しない・表示が崩れる
- ブラウザが最新バージョンであることを確認してください。
- 広告ブロック系の拡張機能が IndexedDB の動作を制限している場合があります。必要に応じて無効化を試してください。

### データが表示されない
- シークレットモード（プライベートブラウズ）を使用している場合、ブラウザを閉じるとデータが消去されることがあります。通常モードでの利用を推奨します。

### 更新が反映されない
- PWAとしてインストールしている場合、一度アプリを閉じて再起動するか、画面上の「更新」通知をクリックしてください。

## プライバシーとセキュリティ
- **Local Only:** 本アプリは、アップデートチェック以外の外部通信を行いません。
- **透明性:** 開発者コンソール（F12）から IndexedDB の中身を直接確認することが可能です。

## カスタマイズ
- **テーマ:** ライトモード / ダークモードの切り替えが可能です。
- **アクセントカラー:** 設定パネルからお好みの色（ブルー、グリーン、オレンジ等）を選択できます。

## 開発者向け情報
- **技術スタック:** Vanilla JS, HTML5, CSS3 (No external libraries).
- **仕様書:** `docs/spec.md` を参照してください。
